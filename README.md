ES UNA CALCULADORA, NO HACE FALTA EXPLICAR MUCHO SOBRE SU PROPÓSITO

SE NECESITA PYTHON 3.11 O SUPERIOR

COMO EJECUTAR:

PRIMERO INSTALAR LAS DEPENDENCIAS:
   ```bash
   pip install PySide6
   ```

LUEGO EJECUTAR LA APLICACIÓN:
   ```bash
   python main.py
   ```

SI QUIERES CREAR UN EJECUTABLE:
   ```bash
   pip install pyinstaller
   pyinstaller --onefile --windowed main.py
   ```
ESTE SE GENERARÍA EN LA CARPETA "dist".
